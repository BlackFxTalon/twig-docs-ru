Support

Support is given through Stack Overflow. If possible, try to reproduce your issue on twigfiddle before asking your question, and add a link to it in your question.

If you want to discuss the enhancement of the library, create an ticket on Github or submit a pull request.

Bug Reports

If you find a bug, you can create a ticket.
